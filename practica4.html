Etiquetas para la Agregación de contenido:

La etiqueta <div> es utilizada para crear un contenedor generico.

La etiqueta <p> es utilizada para escribir párrafos de texto.

Etiquetas Semanticas para Estructura la página web:

La etiqueta <nav> es usada para definir la seccion de navegación de un sitio web.
La etiqueta <main> se usa para definir el contenido principal del documento web.
La etiqueta <section> permite definir una sección del documento web.
La etiqueta <article> es usada para definir contenido independiente de un documento web.
La etiqueta <aside> es usada para definir contenido adicional del documento web.
La etiqueta <header> se usa para especificar la cabecera de un sitio seb.
La etiqueta <footer> es usada para definir el pie de página de un sitio seb.

Etiquetas Semanticas para Texto:

La etiqueta <b> es para poner el texto en negritas.

La etiqueta <strong> es para poner un texto que es importante en la pagina web.

La etiqueta <small> es usada para poner un comentario o contenido que no es esencial para la comprension del documento web.

La etiqueta <cite> es usada para indicar el título de una obra o contenido.

La etiqueta <sub> es usada para representar un subíndice.

La etiqueta <sup> es usada para representar un superíndice.

La etiqueta <mark> es usada para resaltar un texto.

La etiqueta <span> es usada para definir una sección en el documento.

La etiqueta <br> es usada para dar un salto de línea.

Etiquetas para dar Estilo al texto:

Es un párrafo normal con la etiqueta <p>

La etiqueta <b> es para poner el texto en negritas.

La etiqueta <em> es para dar énfasis a las partes importantes de un texto.

La etiqueta <i> es para poner un texto con un estilo en cursiva o italica.

La etiqueta 

<pre>
es usado para poner un texto preformateado.
